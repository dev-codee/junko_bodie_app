## 1. What Is Junko Bodie?

**Junko Bodie Roulette Tournament** is a premium, play-money roulette simulator. Users pay a subscription ($4.99/month or $54.99/year with 7-day trial) to access:
- **Solo play** — American or European roulette against the house
- **Tournament mode** — 6-player tournaments (1 human + 5 bots) with rounds, eliminations, points
- **Season rankings** — Annual championship qualification based on tournament points
- **Profile & badges** — Stats tracking, founder badge, champion badge, elite status

The app has a **dark/gold casino aesthetic** with premium typography (Playfair Display, Crimson Pro, Inter), animated roulette wheel, sound effects, background music, and speech synthesis commentary during tournaments.

## 6. Game Logic Summary

### Betting System (`bets.ts`)
- **15 bet types**: straight (35:1), split (17:1), street (11:1), corner (8:1), sixline (5:1), dozen (2:1), column (2:1), red/black/odd/even/low/high (1:1), trio (11:1), basket (6:1/8:1)
- **Number 37 = 00** (internal representation for American double-zero)
- `BET_MAP`: flat lookup map from betId → BetDefinition
- All bet definitions are generated programmatically (straights, splits, corners, etc.)

### RNG (`rng.ts`)
- `AMERICAN_WHEEL_ORDER`: 38 pockets (0, 00, 1-36)
- `EUROPEAN_WHEEL_ORDER`: 37 pockets (0, 1-36)
- `spinWheel()`: uses `crypto.getRandomValues()` for fair random
- `getNumberColor()`: red/black/green classification
- Result includes: number, displayNumber, color, parity, dozen, column, half

### Payouts (`payouts.ts`)
- `calculatePayouts(bets, spinResult)` → outcomes per bet + totals
- Win = bet amount × payout multiplier; total returned = stake + winnings

### Game Phases (`gamePhases.ts`)
- `BETTING` → `LOCKED` → `SPINNING` → `RESULT` → `RESET` → repeat

### Tournament Points
- 1st place: 1000 pts, 2nd: 100, 3rd: 50, 4th-6th: 0
- Elite status threshold: 5000 season points

## 9. Flutter Project Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| **State Management** | Provider | Simplest, maps closely to React Context pattern used in web app |
| **Navigation** | go_router | Declarative routing with auth guards, similar to Next.js App Router |
| **HTTP Client** | http / dio | Standard, reliable |
| **Audio** | audioplayers | Well-maintained, supports looping, volume control |
| **TTS** | flutter_tts | Replaces Web Speech API for tournament commentary |
| **Animations** | flutter_animate | Replaces Framer Motion, similar API feel |
| **Payments** | In-App Purchase (later) | Required for Play Store / App Store distribution |
| **Backend** | Existing Next.js on Vercel | `https://junkobodieroulette.com/api/*` — no changes needed |
| **Target** | Android + iOS | Both stores |
